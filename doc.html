Get Started

GotsJS simplifies TLS certificate generation and management, offering streamlined workflows and secure encryption options for a seamless experience.

Installation

            
              $ npm install gotsjs

Configuration

Discover how to configure GotsJS to tailor TLS certificate generation and manage TLS connections effectively.

Options Value Default
certificatePath Path to save the generated TLS certificate file. 'certificate.pem'
keyPath Path to save the generated private key file. 'private_key.pem'
outputDir Directory to save the generated certificate and key files. './certs'

Explanation:

  • const gots = require('gotsjs') : This line imports the GotsJS library into our code, allowing us to use its functions and methods.
  • gots.generateAndSaveCert('certificate.pem', 'private_key.pem', './certs') : Here, we call the generateAndSaveCert function from GotsJS, passing in the file names for the certificate and private key, along with the output directory.
  • console.log('Certificate and Private Key generation initiated.') : Finally, we use console.log to print a message indicating that the certificate and private key generation process has been initiated.
            
              const gots = require('gotsjs');
              gots.generateAndSaveCert('certificate.pem', 'private_key.pem', './certs');
              console.log('Certificate and Private Key generation initiated.');
